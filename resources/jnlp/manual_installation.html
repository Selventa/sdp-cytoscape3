Cytoscape Plugin -- Installation

  1. Copy the link to the plugins site
  2. Open Cytoscape
  3. Open the Plugins -> Manage Plugins menu
  4. Go to Settings tab and "Add" download site
  5. Give it a name (like "SDP Plugin Site")
  6. Paste the copied link from step 1 into the URL field
  7. Hit Ok and then select the added download site
    • You may have to resize the "Download sites" control to see the newly added download site
  8. Find the "Available for Install" section
  9. Select KAM Navigator theme and click Install
  10. Select SDP Model Plugin and click Install
  11. Start using the plugins
  12. The KAM Navigator user manual can be found here
  13. The SDP Model Plugin functionality can be found under the SDP menu
